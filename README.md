## ✨ 개요 (Overview)

> **BrickFlow**는 레고 제조 공장에서  
**생산 관리 · 작업 지시 · 품질 검사 · 설비 모니터링** 등  
모든 공정 단계를 **직관적 UI로 효율화**한 **MES 플랫폼**입니다.
